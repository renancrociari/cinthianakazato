# Cínthia Nakazato — Portfolio

Personal landing page for Cínthia Nakazato.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)

## Getting started

```bash
# 1. Install dependencies (only needed once, or after pulling new changes)
npm install

# 2. Start the dev server
npm run dev
```

The site will open at **http://localhost:5173** with live reload — any file change updates the browser automatically.

## Project structure

```
├── index.html          ← main page (entry point)
├── src/
│   ├── css/            ← stylesheets
│   └── js/             ← scripts
├── public/             ← static assets (images, favicons)
└── package.json
```

## Other commands

| Command             | Description                          |
|---------------------|--------------------------------------|
| `npm run build`     | Production build → `dist/`           |
| `npm run preview`   | Preview the production build locally |
